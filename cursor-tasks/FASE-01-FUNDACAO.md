# Fase 1 - Fundação

Objetivo: criar base Next.js, Supabase SSR, auth e layout.

## Escopo

- Next.js App Router.
- TypeScript strict.
- Tailwind/shadcn.
- Supabase SSR clients.
- Middleware de sessão.
- Rotas auth básicas.
- `AppShell` com header/sidebar/footer.
- Design tokens iniciais.

## Não incluir ainda

- Placar.
- Competências.
- RV público.
- CRUDs complexos.

## Critério de aceite

- App abre localmente.
- Login básico funciona ou tem stub seguro se Auth ainda não estiver pronto.
- Layout autenticado renderiza com dados mockados mínimos ou usuário real.
- Typecheck/lint passam.
